# 导购话术生成

输出 `result`：
- `sales_scripts`
- `short_pitch`
- `followup_questions`
- `mobile_reply`

## 手机端话术格式
每套候选默认输出：
- `一句推荐`
- `一句理由`

推荐控制在：
- 单套话术 **20-35 个汉字**
- 全部候选默认最多 **3 套**
- 总输出默认不超过 **8 行**

## 关键规则
- 每套搭配输出一句短推荐 + 一句理由
- 语言贴近门店现场，不要过度文案化
- 优先短句、口语化、可直接复制给顾客或口头说出
- 可按不同顾客偏好输出：显高、显瘦、通勤、松弛、气质、年轻化
- 默认先给短版；长版解释只能作为附加说明，不得占据主输出
