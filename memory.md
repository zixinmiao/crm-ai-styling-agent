# 预置记忆

## 目标问题
- 商品资料分散在 PDF、品牌图册、导购收藏图和库存表中，无法快速转成可执行搭配建议
- 导购在客人试衣时缺少“基于当前在库商品”的候选搭配
- 门店缺少自己的搭配知识库，优秀穿搭经验难复用

## 成功标准
- 建立可查询的商品知识库：款号、品类、颜色、尺码、价格、库存、风格、版型、材质、适合身形
- 建立可查询的搭配知识库：look 风格、单品组成、色彩关系、适合身形、适合场景、推荐理由
- 输入一件商品或一张商品图时，可输出 3-5 套门店可执行搭配候选

## 推荐知识载体
- 飞书多维表格：适合结构化商品库 / 搭配库 / 库存库
- 飞书文档：适合沉淀搭配方法论、风格说明、商品培训摘要
